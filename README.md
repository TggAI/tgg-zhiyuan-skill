# 糖果果志愿同行 · 高考志愿填报参谋 (tgg-zhiyuan-skill)

一个高考志愿填报辅助 **skill**。它扮演一位"志愿填报参谋"：先让考生做方向测评问卷，再按分数查本省一分一段表反推位次，结合方向画像与偏好收窄方向，到权威官网查院校专业、排冲稳保梯度，校验选科，最后产出带来源、可复核的建议。

> ⚠️ 所有产出均为信息整理与参考，标注"待复核"、不输出确定性结论，最终志愿由考生与家长、老师共同决定。

## 怎么用

1. **导入 skill**：把整个文件夹放进支持 skill 的 AI 环境（如豆包）的 skills 目录，让 AI 加载本 skill。
2. **做问卷**：让 AI 把 `assets/questionnaire.html` 输出出来，在页面里填（省份、选科、分数 + 13 道方向测评题），做完点【复制全部规划数据】。
3. **粘回对话**：把复制的内容发给 AI，它就接着按下面的流程帮你规划。
4. **按需取用**：不想做问卷，可以直接告诉 AI 你的省份 / 选科 / 分数；只想查位次、只想校验选科，也都能单独问。

## 流程（参谋会怎么帮你）

```
阶段0  问卷        做方向测评 → 得出方向画像 + 推荐指数榜 + 偏好
阶段1  共情        看分数说句真诚的话，再干活
阶段2  查位次      用分数查本省一分一段表 → 得全省位次（带来源）
阶段3  收窄方向 ⏸  用 位次 + 推荐专业 + 省份偏好，定下"查哪些省 / 专业 / 档次"，停下确认
阶段4  查院校      去权威官网（考试院 / 阳光高考 / 院校官网）查录取位次、招生、就业
阶段5  冲稳保      按 考生位次 vs 院校录取位次，排 冲 / 稳 / 保
阶段6  选科校验    核对选科能不能报目标专业
阶段7  交付 ⏸      出 Excel 对比表 + Word 建议书，列出待核实项，停下请家长 / 老师复核
```

`⏸` = 关键步骤，参谋会停下来请你（尤其家长 / 老师）确认，不替你拍板。

## 三条红线

1. **数据可查证**：每个数字都带来源，查不到就写"未查到"，绝不编。
2. **结论不下定论**：给带依据的选项和梯度，不替任何人做最终决定。
3. **关键步骤停下确认**：收窄方向、最终志愿表都停下请用户复核。

## 目录结构

```
SKILL.md                         技能主文件（流程、红线、附录）
assets/questionnaire.html        方向测评问卷（含推荐指数、可导出规划数据）
references/一分一段表/<省份>/      各省一分一段表 CSV（按省份分目录）
```

## 数据说明

- 一分一段表覆盖 31 个省级单位、54 份 CSV（绝大多数为 2026 年，海南为 2025 年）。
- CSV 列：`省份,年份,科类,分数段,分数,同分人数,累计人数,来源网页,图片页码`，其中 `累计人数` 即该分数对应的全省位次。
- 本仓库为精简版：OCR 源图（png / jpg）、xlsx、构建脚本未包含。
